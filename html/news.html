Current news (last update: 21 Jul 2000)

Version 0.97 of Gambit is under heavy development.  We have updated the graphical user interface to work with version 2.2 of the wxWindows library.  This has allowed us to greatly streamline the interface, and also to support the GTK windowing toolkit to give us a stable, freely-available platform for use on Linux systems. 

We have not yet set a target date for the public release of version 0.97.  We will first wait for the final production release of version 2.2 of wxWindows to make sure the graphical interface is as stable as possible.  Check back for updates; our current best guess is that 0.97 will be ready in the fall of 2000.

March 2000: Gambit 0.96.2 released

Bug fixes:
  • Bug in File select dialog fixed
  • Fixed bug in installation script

February 2000: Gambit 0.96 released

Feature list:
  • New polynomial code for computation of all Nash equilibria of n-person games
  • Added clique code (thanks to Bernhard von Stengel) for characterizing the complete set of Nash equilibria in non generic two person games
  • Command line editor added to GCL
  • Improved tableau and linear programming code for speeding up Rational precision computations
  • New Number Class added to support mixed Floating point and Rational computaations
  • Added exception handling to stabilize code
  • Fixed many bugs from release .94
