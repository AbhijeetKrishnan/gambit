Current news (last update: 5/29/2000)

Version 0.97 of Gambit is under heavy development.  We have update the graphical user interface to work with version 2.x of the wxWindows library.  This has allowed us to greatly streamline the interface, and also to support the GTK windowing toolkit to give us a stable, freely-available platform for use on Linux systems. 

We have not yet set a target date for the public release of version 0.97.  We will first wait for the final production release of version 2.2 of wxWindows to make sure the graphical interface is as stable as possible.  Check back for updates; our current best guess is that 0.97 will be ready in the fall of 2000.
