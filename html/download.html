Gambit is available for free download and use.  Please see our copyright notice if you are interested in using Gambit as part of your own free or commercial application.

System Requirements

Gambit is designed to be functional on a wide range of Unix and Microsoft Windows (Windows 95 or newer) systems.  To use the graphical user interface under Unix (X Windows), you will need to have a sufficiently new version of either the GTK or Lesstif window interface libraries (available for free from their respective sites).

If you wish to compile Gambit on your own, you will need a sufficiently new C++ compiler
that supports templates and exceptions.  For Unix users, we write code that compiles and works under the latest version of GCC.  For Windows users, our code has been tested under Borland C++ 5.02, and the free command-line tools (Borland C++ 5.5) available on Borland's website.  We would be grateful to hear of experiences (or better yet, makefiles and/or patches!) using other compilers.

Binary (Executable) Distributions

We have prepared executable versions of the Gambit applications and libraries for several common platforms. 

The executable distributions include the following:

  • Graphical user interface
  • Graphical interface to command language interpreter
  • Text (console)-based interface to command language interpreter
  • PXI, an auxiliary plotting application
Executable distributions are available for the following platforms:
  • Executables for Microsoft Windows (95/98/2k/NT) (.zip file)
  • Executables for RedHat Linux:
    • Compiled for GTK (requires GTK version ???) (tar'ed, gzip'ed file)
    • Compiled for Lesstif (requires Lesstif version ???) (tar'ed, gzip'ed file)
  • Executables for Solaris (requires Lesstif version ???) (tar'ed, gzip'ed file)
