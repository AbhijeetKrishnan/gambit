Gambit is available for free download and use.  Please see our copyright notice if you are interested in using Gambit as part of your own free or commercial application.

System Requirements

Gambit is designed to be functional on a wide range of Unix and Microsoft Windows (Windows 95 or newer) systems.  To use the graphical user interface under Unix (X Windows), you will need to have a sufficiently new version of either the Motif or Lesstif window interface libraries (Lesstif is available for free from www.lesstif.org.)

If you wish to compile Gambit on your own, you will need a sufficiently new C++ compiler that supports templates and exceptions.  For Unix users, we write code that compiles and works under the latest version of GCC.  For Windows users, our code has been tested under Borland C++ 5.02. We would be grateful to hear of experiences, or better yet, to receive makefiles and/or patches for using other compilers or systems.

Binary (Executable) Distributions

We have prepared executable versions of the Gambit applications and libraries for several common platforms. 

The executable distributions include the following:

  • Graphical user interface
  • Command language interpreter
  • PXI, an auxiliary plotting application
Executable distributions are available for the following platforms:
  • Executables for Microsoft Windows (95/98/2k/NT) (.zip file)
  • Executables for Linux (Built with RedHat Linux 6.0, g++ 2.95.2 and Motif 2.1.0)
  • Executables for Solaris (Built with SunOS 5.7, g++ 2.95.2 and Motif 2.1.10)

Source Code Distributions

The complete source code for Gambit is also available:
  • Gambit source code for MS Windows (.zip file)
  • Gambit source code for Unix (.zip file)
