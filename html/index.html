Gambit is a library of game theory software and tools for the construction and analysis of finite extensive and normal form games.  Gambit is designed to work on both Microsoft Windows (95/98/NT) and Unix (Linux, Solaris, and others) platforms.  Gambit includes:
  • A graphical user interface (GUI), based upon the wxWindows library, providing a common look-and-feel across platforms.
  • The Gambit Command Language (GCL), a scripting language with built-in primitives for constructing and analyzing games.
  • A library of C++ source code for representing games, suitable for use in other applications.
The Gambit source code, and precompiled executables, are freely available, and may be used in other applications.  Please see our copyright notice for details.
