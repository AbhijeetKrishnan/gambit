EnumMixed

Finds all Nash equilibria for a two person game. More precisely, it finds the set of extreme points of the components of the set of Nash equilibria. The procedure is to enumerate the set of complementary basic feasible solutions (see eg. /citeMan:64.)

Limitations: Only works for two-person normal form games.

The following parameters can be set:

Stop After:
Specifies the maximum number of equilibria to find. The default is zero, which means that all equilibria are found. To check if there is a unique Nash equilibrium, one could set this parameter to 2.
precision:
Select Rational for exact computations, Float for speed. See the section on Data Types for more information.
