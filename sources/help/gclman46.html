Actions

Actions[infoset->INFOSET] =: LIST(ACTION)
Returns a list of all actions in infoset.

Definition: 

  Actions[infoset,Support[Game[infoset]]]

Actions[efg->EFG] =: LIST(LIST(LIST(ACTION)))
Returns a nested list of actions in the game, whose (i,j,k) entry is action k in information set j of player i.

Definition: 

  Actions[Infosets[Players[efg]]]

Actions[s->EFSUPPORT] =: LIST(LIST(LIST(ACTION)))
A nested list of actions for the support s in the extensive form game.

Definition: 

Actions[Infosets[Game[s]],s]
