ActionNumber

ActionNumber[action->ACTION, sup->EFSUPPORT] =: INTEGER 
ActionNumber[action->ACTION, basis->EFBASIS] =: INTEGER
Returns the number assigned to action in the support sup (or basis basis). For each information set, actions are numbered from 1 to the number of actions at the information set. If an action is not in the corresponding support or basis, it is assigned a number of 0.

See also: Actions.
