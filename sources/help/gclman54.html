AllSubgamePerfect

AllSubgamePerfect[e->EFG, {stopAfter->0}, {nTries->0}, 
                  {precision->Float}] =: LIST(BEHAV)
Searches for all subgame perfect Nash equilibria of the extensive form game e. Returns a list containing the equilibria found.

Definition: 

  ms:=MarkedSubgames[e];MarkSubgames[e];
  If[NumPlayers[e] = 2, 
    b:=EnumMixedSolve[e,True,0,precision],
    b:=PolEnumSolve[e,False,stopAfter]
  ];
  MarkSubgames[ms];b
