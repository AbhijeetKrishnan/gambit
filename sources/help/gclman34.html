AddAction

AddAction[support->EFSUPPORT, action->ACTION] =: EFSUPPORT 
AddAction[basis->EFBASIS, action->ACTION] =: EFSUPPORT
Adds action to the support. If the action is already in the support, no change is made. A copy of the modified support is returned. It is an error if support and action are not from the same extensive form game.

See also: RemoveAction (REF NOT FOUND), Support (REF NOT FOUND).
