Assign

Assign[name->TEXT, value<->T*] =: T
for all types T

Creates a new variable name, with return value of value. Returns the value assigned. If name is already a defined variable in the current scope, then if it is of type T, it is modified to have return value of value. If it is of a different data type, an error results.

Short form: x := value

Note: For the short form, quotes around name are not required, and the function is not listable.

See also: Clear (REF NOT FOUND), UnAssign (REF NOT FOUND).
