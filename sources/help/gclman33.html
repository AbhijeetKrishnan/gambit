ActionValue

ActionValue[profile->BEHAV, action->ACTION] =: NUMBER
Returns the expected payoff to a player (who is in control of the information set including action) of choosing action when all players are following profile. If the value is not well-defined, the null value is returned. It is an error if profile and action are not from the same extensive form game, or if action is an action at an information set owned by the chance player.

See also: ActionValues, Belief (REF NOT FOUND), InfosetProb (REF NOT FOUND), NodeValue (REF NOT FOUND), RealizProb (REF NOT FOUND).
