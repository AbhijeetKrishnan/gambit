Standard Solution Dialog

This dialog uses standard settings to compute various types of equilibria. For more information on the settings used, see the section on NFG Standard Solutions for the normal form or EFG Standard Solutions for the extensive form.

Figure 4.7: Standard Solution dialog for normal form games

Picture
Figure 4.8: Standard Solution dialog for extensive form games

Select one choice in each box. Not all combinations are available for all games, in which case, some choices will be greyed out. For example Rational computations are only supported on certain algorithms on two person games, Perfect equilibria can only be computed for two person games.
