Actions

Actions[infoset->INFOSET*, support->EFSUPPORT] =: LIST(ACTION) 
Actions[infoset->INFOSET*, basis->EFBASIS] =: LIST(ACTION)
Returns the list of actions available at infoset in support or basis. If infoset is the null information set, the empty list of actions is returned. It is an error if infoset and support are not from the same extensive form game.

See also: DeleteAction (REF NOT FOUND), InsertAction (REF NOT FOUND).
