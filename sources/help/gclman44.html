ActionProbs

ActionProbs[profile->BEHAV]=:LIST(LIST(LIST(NUMBER)))
Returns a nested list of probabilities for each action under profile. The (i,j,k)th entry is the probability of player i, information set j, action k, where the index numbers correspond to the output of Players, Infosets, and Actions, respectively.

See also: ActionProb, SetActionProbs (REF NOT FOUND).

Definition: 

ActionProb[profile,Actions[Game[profile]]]
