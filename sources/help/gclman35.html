AddMove

AddMove[infoset->INFOSET, node->NODE] =: NODE
Places a terminal node into the information set infoset. The result is node becomes a decision node, with number of branches equal to the number of actions in infoset. New child nodes are created as successor nodes to the actions in the information set. The child which succeeds the first action is returned. It is an error if node and infoset are not from the same extensive form game, or if node is not a terminal node.

See also: InsertMove (REF NOT FOUND), DeleteMove (REF NOT FOUND), NewInfoset (REF NOT FOUND).
