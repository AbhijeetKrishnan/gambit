ActionValues

ActionValues[profile->BEHAV]=:LIST(LIST(LIST(NUMBER)))
Returns a list of expected payoffs of each action (to the player who chooses that action). The null value is returned for actions for which this value is not well-defined. The (i,j,k)th entry is the expected payoff to player i at information set j, of action k, where the index numbers correspond to the output of Players, Infosets, and Actions, respectively.

See also: ActionValue, Belief (REF NOT FOUND), InfosetProb (REF NOT FOUND), NodeValue (REF NOT FOUND), RealizProb (REF NOT FOUND).

Definition: 

ActionValue[profile,Actions[Game[profile]]]
